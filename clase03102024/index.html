XAMPP

Para que sirve XAMPP

crear una distribución fácil de instalar para desarrolladores que se están iniciando en el mundo de Apache.

¿Qué puedo hacer con XAMPP? XAMPP es una herramienta muy popular para montar servidores locales. Con XAMPP puedes crear entornos de desarrollo y probar páginas web antes de publicarlas en un hosting.
