CODIGO DE PROGRAMACION

La programación es el proceso de escribir instrucciones para que una computadora pueda realizar una tarea específica. Se trata de adaptar el potencial de las computadoras a las necesidades de los usuarios y empresas, para generar procesos automáticos que abordan desde necesidades laborales hasta la creación de videojuegos.
